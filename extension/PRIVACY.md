# Privacy Statement

ASCAP Registration Triage is an independent browser-assisted metadata triage tool. It is not affiliated with, endorsed by, or operated by ASCAP.

## Data Stored Locally

The extension stores the following data locally in Chrome extension storage:

- Work metadata entered by the user.
- Captured ASCAP public repertoire candidate records.
- Local analysis results.
- Visual theme preference.

## Data Transmission

The extension does not send user data to a hosted application server and does not require a local backend.

When the user asks the extension to search or analyze, it may:

- Open or interact with ASCAP public repertoire pages.
- Query public music metadata sources such as MusicBrainz, Wikidata, and Wikipedia for advisory songwriter reference evidence.

Those public reference requests may include the work title, performer, writer clues, publisher clues, or captured ASCAP candidate writer names needed to identify likely public songwriter credits.

## Data Not Collected

The extension does not collect, store, or transmit:

- ASCAP member portal credentials.
- Private account data.
- Payment information.
- Browser history outside the active ASCAP page used by the extension.
- Bulk page content from arbitrary websites.

## User Control

Users choose when to open ASCAP searches, fill ASCAP fields, capture visible ASCAP results, clear candidates, copy reports, and copy Work IDs.

Public writer reference data is advisory only. It supports review and does not replace ASCAP verification.

## Limited Use

Information handled by ASCAP Registration Triage is used only to provide and improve the extension’s stated metadata-triage features. It is not sold, used for advertising, used to determine creditworthiness, or transferred for unrelated purposes. Any transfer to ASCAP, MusicBrainz, Wikidata, or Wikipedia occurs only as necessary to provide a feature requested by the user.

The use of information received from Google APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements.
